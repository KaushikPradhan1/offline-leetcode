Offline LeetCode
Offline LeetCode is a powerful, 100% offline-first Visual Studio Code extension designed for data structures and algorithms (DSA) practice. Solve thousands of problems directly inside your editor without requiring any active internet connection or external web browser.

✨ Key Features
3,000+ LeetCode Problems: Built-in offline dataset containing problem descriptions, difficulties, categories, and test cases.

100% Offline Execution: Run code locally on your machine. No cloud dependencies, tracking, or network overhead.

Multi-Language Support:

Python 3 & JavaScript: Local auto-evaluation against structured test cases with pass/fail feedback.

C, C++, & Java: Local compiler integration (g++, gcc, javac) for offline playground execution.

Persistent Progress Tracker: Tracks your solved questions, accuracy percentages, and status (Solved / Attempted) directly via VS Code globalState.

Sleek VS Code Native UI: Professional modern toolbar, responsive layout, and custom-styled language selection.

🚀 Getting Started
Install the extension from the VS Code Marketplace or load it locally via .vsix.

Click the Offline LeetCode icon in your Activity Bar (sidebar).

Expand any category (e.g., Arrays & Hashing, Trees, Dynamic Programming).

Click on any problem to open the interactive problem panel and split-screen code editor.

🛠️ How to Use & Test
Select a Language: Use the dropdown selector in the top-right toolbar of the problem panel.

Write Your Solution: Implement your code using the provided class/function signature.

Run Code: Click ▶ Run Code to test your implementation locally.

Track Progress: Once all test cases pass (100% accuracy), the problem is automatically marked as Solved with a green badge in your sidebar.

⚙️ Requirements
Node.js & Python 3: Required for running the extension host and evaluating Python/JavaScript test scripts.

Local Compilers (Optional): To execute C, C++, or Java locally offline, ensure you have g++, gcc, or javac installed and added to your system's PATH.
